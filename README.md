# Big_data_clustering_electricity_consumption

This document is an explanatory report of the clustering project carried out for the validation of the option “Building and Developing Big Data Models”, as part of the “Expert in Computer Science and Information Systems” training program.

The code can be executed in a Python virtual environment; however, since the dataset is not publicly accessible, it cannot be shared. As a result, the code cannot be executed outside the oral presentation or outside the organization that owns the project data.

For more details, please refer to the accompanying PDF document. In particular, Section 3 contains a subchapter that explains the steps followed to create the Python environment that served as the foundation for the development of this project.
